# 网球培训机构运营全局指南

> 本文档为机构运营的全局参考框架，涵盖战略定位、产品体系、教练管理、获客留存等核心模块。所有执行细节以本文档为准，如有调整请直接修改对应章节。

---

## 一、战略定位

**核心定位：** 以青少年为压舱石，成人为现金流，装备与服务为增值层，赛事为增长引擎。

**客群比例目标：**

| 客群 | 目标占比 | 核心价值 |
|------|---------|---------|
| 青少年竞技 | 50-55% | 高客单、高粘性、强口碑传播 |
| 成人技术提升 | 35-40% | 稳定现金流 |
| 企业团课 | 5-10% | 机缘型，有则接，不主动押注 |

---

## 二、基础设施：NTRP 段位体系

### 核心逻辑

以国际通用 NTRP 作锚点，内部加 A/B 细分制造进步感。对外说得出口（"我是 3.0"），对内进步感细化（每 1-2 个月可升一档）。

### 段位结构

```
2.5B → 2.5A → 3.0B → 3.0A → 3.5B → 3.5A → 4.0B → 4.0A
```

### 各段位画像参考

| 段位 | 大概水平描述 |
|------|------------|
| 2.5B | 刚入门，能过网，动作有雏形但发力靠手臂，缺乏稳定性 |
| 2.5A | 有转体发力意识，正反手能稳定对打，准备冲 3.0 |
| 3.0B | 能基本对打，正反手有雏形，发球能进，移动能跑动接球 |
| 3.0A | 能主动控制落点，开始有战术意识，准备冲 3.5 |
| 3.5B | 有对抗能力，能执行简单战术，覆盖全场基本位置 |
| 3.5A | 能根据比分和对手调整打法，发球有战术意图，准备冲 4.0 |
| 4.0B | 能在业余比赛中竞争 |
| 4.0A | 能稳定赢 3.5 级对手，有参加联赛的实力 |

### 晋级评估标准（五维，达 4/5 项即可晋级）

| 维度 | 评估内容 |
|------|---------|
| 正手稳定性 | 对打回合数 + 落点控制能力 |
| 反手稳定性 | 稳定性 + 主动进攻能力 |
| 发球 | 成功率 + 是否有战术意图 |
| 移动 | 覆盖范围 + 预判能力 |
| 战术意识 | 场上决策与执行能力 |

**青少年附加维度：** 比赛心理与竞争意志，单独评估，供家长参考。

### 晋级节奏

- 档内晋级（B → A）：约 1-2 个月
- 级间晋级（如 2.5A → 3.0B）：约 3-6 个月
- 目标：每位学员每年可感受到 2-3 次晋级

### 执行方式

教练在日常训练中持续观察打分，累计 2-3 次训练确认稳定达标后晋级，不做一次性考试，避免某天状态好蒙混过关。

## 三、赛事体系

### 赛事节奏

| 类型 | 频次 | 规模 | 核心目的 |
|------|------|------|---------|
| 季度小赛 | 每季度一届 | 20-30 人，半天 | 维持活跃度，低成本运营 |
| 半年大赛 | 每半年一届 | 50-80 人，全天 | 品牌展示，对外拉新 |

### 赛制结构（三方向整合）

**上午：团队循环赛**
- 按 NTRP 混搭组队（高中低段位搭配，每队 3 人）
- 小组循环积分制，每人至少打 3 场
- 输赢有队友分担，心理压力稀释

**下午前半段：个人技术挑战赛**
- 发球成功率挑战、定点落点挑战、最长对拍回合挑战
- 无输赢概念，只有自己 vs 自己的数据
- 成绩记入段位档案，下届可对比进步

**下午后半段：名次决赛（自愿参与）**
- 团队赛积分前几名进入
- 不想参与的人自由活动，不产生被排斥感

### 奖项设计（覆盖 40-50% 参赛者）

| 奖项 | 产生方式 | 奖品参考 |
|------|---------|---------|
| 团队冠亚季军 | 积分排名 | 装备礼包 |
| 技术挑战各项第一 | 客观数据 | 手胶 / 网球套装 |
| 最佳进步奖 | 教练提名，对比上届数据 | 训练衣 |
| 最佳体育精神奖 | 全员投票 | 装备小礼包 |
| 最佳搭档奖 | 对拍回合最长组合 | 双人礼包 |

### 赛后诊断卡（每位参赛者必发）

```
【×× 的本届赛事小结】

打得好的地方：
1. _______________
2. _______________

一个值得改进的点：
_______________

下阶段训练建议：
_______________

教练签名：___
```

### 赛前预期管理

赛前一周在学员群发通知，明确本次比赛目的不只是分出高下，更重要是让每个人看清真实状态，找到下一步练球方向。帮助学员将心理预期从"我要赢"调整为"我来照镜子"。

### 对外开放名额（拉新核心动作）

- 每届大赛留 10-15 个外部名额，只能由现有学员邀请带入
- 外部选手参赛后：教练主动点评 + 送 30 天有效体验课券 + 30 天内跟进一次

### 赛事与段位联动

- 比赛成绩计入段位积分
- 赢得跨段位对手可触发晋级挑战资格
- 颁奖时同步公布本届晋级学员名单

### 内容产出（比赛当天必做）

- 专人负责拍照和短视频，重点拍精彩球、颁奖、学员笑脸
- 当天晚上剪 60 秒混剪发到学员群
- 每位学员单独发一张个人高光照片

## 四、装备合作线

### 角色定位

机构不是零售商，是**流量入口 + 信任背书**。学员对机构的信任是核心筹码，不是订单量。

### 品类独家原则

| 品类 | 合作方式 |
|------|---------|
| 球拍类 | 独家品牌 A |
| 网球 / 手胶 | 独家品牌 B |
| 服装 / 鞋 | 独家品牌 C |
| 球包 | 与服装品牌合并或独立品牌 D |

同类目保持独家，不引入竞品，提升对每个品牌的谈判价值。

### 五个利润点

| 利润点 | 说明 |
|--------|------|
| 差价分润 | 基础利润，不作主要依赖 |
| 陈列 / 曝光费 | 季度固定市场费用，参考 3000-8000 元 / 品牌 / 季 |
| 赛事冠名赞助 | 现金或装备置换，随赛事影响力增长 |
| 段位晋级礼包 | 升段时推荐适配装备，顺势转化 |
| 课程捆绑销售 | 买拍送体验课，装备销售与获客双赢 |

### 消耗品复购引擎

- **训练用球统一采购**：费用摊入月费，品牌走量稳定可预测
- **手胶订阅包**：每月固定配送，随课费自动续费
- **穿线服务**：作为增值服务，制造定期到店触点

### 穿线服务定价

| 服务层级 | 内容 | 定价参考 |
|---------|------|---------|
| 基础穿线 | 客户提供网线，只收手工费 | 50-80 元 |
| 标准穿线 | 含普通网线 | 120-180 元 |
| 高阶穿线 | 含高端网线 + 张力建议 | 200-350 元 |

配套建立学员穿线档案（球拍型号 / 线型 / 张力 / 日期 / 建议下次时间），到期主动提醒。

### 装备销售边界原则

教练不参与装备销售，由运营负责。所有推荐基于学员段位和真实需求，不在训练课上主动推销，避免学员产生"这机构老想卖东西"的反感。

### 品牌合作协议框架（引入新品牌时使用）

| 谈判要素 | 机构诉求 | 品牌诉求 |
|---------|---------|---------|
| 独家类目 | 锁定品类不引入竞品 | 排他保护 |
| 固定曝光费 | 季度 / 年度市场费用 | 可量化曝光回报 |
| 赛事赞助额度 | 每届装备 / 现金支持 | 品牌露出、照片授权 |
| 学员专属价 | 进货折扣 + 差价空间 | 走量 + 精准客群触达 |

## 五、青少年线

### 核心逻辑：做家长，不是做孩子

| 需求层次 | 家长真实诉求 | 机构应对 |
|---------|------------|---------|
| 表层 | 孩子学会打球 | 教学质量基础保障 |
| 中层 | 孩子有进步、有成就感 | 段位体系 + 晋级证书 |
| 底层 | 我的时间被解放，孩子是安全的 | 轻托管 + 亲子套餐 |

### 三个核心产品

**① 常规课 + 月度进度反馈**

每月给家长发一条语音，三句话：这个月练了什么、哪里有进步、下月重点攻什么。两分钟，成本为零，留存效果极强。

**② 寒暑假集训营**

```
5 天结构：
  上午：技术训练（2 小时）
  下午：对抗练习 + 游戏化训练（1.5 小时）
  每天结束：教练点评 5 分钟
  第五天：小型内部赛 + 颁奖仪式 + 家长观礼
```

- 定价参考：1500-3000 元 / 人 / 5 天
- 老带新：现有学员带同学来打八折，老学员获课时奖励
- 起步建议：先在现有学员内部试点，成功后再对外开放

**③ 晋级证书**

升段发正式证书，含机构 logo、教练签名、日期。成本接近零，家长发朋友圈概率高，触达的是同龄孩子家长——最精准的潜在客户。

### 轻托管方案

**全天时间结构：**

```
08:30  家长送达，签到
09:00-11:00  上午训练（教练负责）
11:00-12:30  休息 + 午餐（助理看管）
12:30-13:30  午休 / 自由活动（指定区域，助理看管）
13:30-15:30  下午训练（教练负责）
15:30-16:00  等待家长接送
16:00  家长接走，签退
```

**三个必须到位的基础设施：**

1. **责任协议书**：入营前签署，明确健康状况、紧急联系人、责任边界、意外处理流程
2. **意外险**：每人每天几元，摊入营费，发生意外有理赔保障
3. **专职助理**：非教练角色，体育专业兼职大学生即可，负责非训练时段的陪同和观察

**家长安心方案（代替低质直播）：**

- 每天三次节点推送：签到照、午餐状态、训练结束（助理用手机拍发家长群）
- 主训练区固定摄像头（萤石云等家用产品）接入私密链接，家长想看随时可看
- 每天结束时助理给每位家长发 30 秒语音小结

**隐私说明（写入报名协议）：** 训练区域录像用于家长安全查看及教学回顾，不对外传播。

### 家长预期管理

入学第一个月做一次家长面谈，给出具体进度预期：

> "×× 现在真实水平是……，三个月后预计达到……，半年后大概是……"

锚定预期，防止 3-6 个月后因预期差引发投诉或流失。

## 六、成人线

### 核心认知

成人选网球是反向筛选，来找教练的核心诉求是**技术提升或高质量个人锻炼**，不是社交。

成人留存绑定的是教练，不是机构——这个风险在教练管理层面对冲，不能靠社群运营掩盖。

### 弹性课时包

| 产品 | 内容 | 定价逻辑 |
|------|------|---------|
| 月卡 | 4 节 / 月，固定时间 | 最高单价，入门产品 |
| 季度包 | 12 节，3 个月有效，弹性预约 | 九折，锁定 3 个月 |
| 半年包 | 24 节，6 个月有效 | 八折，送 2 次月度球局参与资格 |

每月允许请假一次顺延补课，不做"请假作废"的死规定。

### 晋级仪式感（成人版）

不发证书，在月度球局上由教练当众宣布：

> "今天要说一下，×× 已经稳定达到 3.0A 水平，可以上进阶班了。"

圈子里的公开认可，比证书更有效。

### 亲子套餐

**核心设计：同步训练时段**

```
家长送孩子 → 家长同步上成人课（相邻球场，视线内可见孩子）→ 一起回家
```

时间没多花，家长多了一节课的价值，安全感也有了。

**定价结构：**

| 套餐 | 内容 | 折扣 |
|------|------|------|
| 亲子月卡 | 家长 + 孩子各 4 节 / 月 | 九折 |
| 亲子季度包 | 各 12 节 / 季，送家庭段位评估一次 | 八五折 |
| 亲子年卡 | 各 48 节 / 年，送寒暑假营优先报名权 | 八折 |

**情感锚点：「能和孩子一起打球」**

入学时告诉家长：

> "按这个节奏练，大概一年后你们可以在场上对打了。"

每季度更新进度感：

> "您现在 2.5A，孩子已经到 2.5B 了，预计再两个季度可以对打。"

这个共同目标是续费最强的长期驱动力。

## 七、教练激励与管理

### 核心风险认知

教练是最大资产，也是最大风险。成人学员与教练之间存在强人际粘性（类似理发师与客户），教练离职大概率带走部分学员，这个风险无法完全消除，只能对冲。

### 三层收入结构

| 层级 | 内容 | 目的 |
|------|------|------|
| 低底薪 | 2000-3000 元 / 月 | 基础安全感，换取稳定性 |
| 课时提成（阶梯制） | 超量部分比例上涨 | 激励勤快，产生内部正向竞争 |
| 留存奖金（季度结算） | 续费率 80% 以上触发奖励 | 让教练利益与学员留存直接挂钩 |

### 晋升体系

```
实习教练 → 助理教练 → 正式教练 → 高级教练 → 首席教练
```

晋升标准透明：学员续费率 + 教学评分 + 机构活动投入度。

**首席教练职责：**
- 制定和维护 NTRP 段位评估标准
- 定期观课，给其他教练反馈
- 新教练入职培训

### 分散教练风险

- 单个教练负责学员不超过机构总量的 25-30%
- 定期安排不同教练带同一班次，学员认识多个教练
- 所有对外内容突出机构品牌，教练是执行者而非主角
- 学员群群主是机构账号，不是教练个人账号

### 教练文化

- 每月一次一对一谈话（不是考核，就是聊状态和想法）
- 公开认可好表现（在学员群表扬）
- 让教练参与段位标准和赛事设计，增加归属感

### 教练想自立的处理

不强硬阻止，提供挂靠合作框架：

> "你想创业我支持，我们可以谈挂靠合作——你在机构品牌下独立运营，我提供场地、品牌、学员资源，你分成 X% 给机构。"

把潜在竞争者变成加盟合伙人。

## 八、获客体系

### 结构化转介绍机制

规则简单，兑现及时：

> 现有学员推荐新学员成功报名，推荐人获 X 节课时奖励，被推荐人首月九折。

### 短视频内容

每周一条 90 秒技术纠错视频，针对业余球员最常见动作错误。手机拍，不需要精良剪辑，持续产出建立搜索流量和专业认知。

### 私域运营原则

80% 内容对学员有价值（技术 tips、比赛信息、进步展示），20% 才是机构信息。比例反过来私域会慢慢死掉。

### 企业团课（机缘型）

通过现有成人学员中的企业员工切入，提供企业福利团课方案。不主动押注，有合适入口则接。

## 十、执行时间线

### 第 1-3 个月：建基础

- 制定 NTRP A/B 段位标准，和教练开会确认
- 整理现有学员档案，完成初始段位评估
- 青少年家长群建立，开始月度进度反馈
- 装备合作品牌引入，穿线服务启动
- 教练薪酬结构改革，签新协议

### 第 4-6 个月：跑飞轮

- 第一届品牌化段位赛（含外部开放名额）
- 推出季度包和亲子套餐
- 寒暑假营试运营（现有学员内部试点）
- 短视频内容开始持续产出
- 结构化转介绍机制正式上线

### 第 7-12 个月：拉新提速

- 季度赛节奏固定，半年大赛升级
- 轻托管产品对外开放
- 口碑转介绍形成自然流量
- 根据教练产能评估是否扩招
- 装备合作品牌根据体量重新谈判条件

## 附：核心逻辑总结

**段位体系**是骨架，让所有人看见自己的进步。

**赛事**是引擎，对内制造荣誉感，对外打开获客口子。

**青少年**是压舱石，客单高、粘性强、抗教练流失。

**亲子套餐**是差异化，卖的是家庭共同目标和时间效率。

**教练管理**是风险控制，分散集中度、绑定利益、建立归属感。

**装备与服务**是增值层，把学员的消费场景锁在机构内。

六个模块互相咬合，缺一个都会让飞轮转得慢。
